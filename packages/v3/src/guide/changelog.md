# 更新日志

::: tip 发布周期
- 主版本号：含有破坏性更新和新特性，不在发布周期内。
- 次版本号：带有新特性的向下兼容的版本。
- 修订版本号：日常 bugfix 更新
:::


## v3.0.4
`2021-10-09`

> 需升级前端文件 `v2.0.5`

- 增加应用/版本设置专属的全局参数headers与parameters。
- 配置docs的path支持使用`${lang}`变量指定当前语言。
- 接口注解的调试时事件Before/After，支持ref引入公共注解。
- 修正拖动浏览器宽度，首页异常问题
- 修正接口注解标题多语言解析异常问题
- 修正点击更新，左侧菜单无法正常展开并选中当前的问题
- 修正刷新页面时，当前应用、md文档打开异常问题
