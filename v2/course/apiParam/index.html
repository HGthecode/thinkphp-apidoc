# 各种参数类型的注解教程

提示

本文注解用@Apidoc\Param（请求参数）举例，如果是返回参数，使用@Apidoc\Returned即可

使用apidoc对接口的请求参数Param与响应参数Returned，根据所需的参数类型，可参考以下示例实现你的接口参数的注解

# string 字符串

# 注解：

/**
 * @Apidoc\Param("name", type="string",desc="姓名" )
 */
1
2
3

# json效果：

name:"";
1

# 文档效果：

名称 类型 必填 默认值 说明
name string 姓名
